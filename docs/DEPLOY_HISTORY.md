# Pricer3D 部署历史

> 从 VERSION 文件迁移。完整变更详情请参考 git log。

## 0.45.0 (2026-07-04)
- 完整明暗主题切换系统 — CSS变量方案 + .dark类切换
- :root定义浅色, .dark定义深色变量; design-tokens.css覆盖所有语义色
- theme.js: 纯文本切换按钮, localStorage持久化, 0.3s过渡动画
- head.html: 移除废弃的dark-theme style元素
- 覆盖: 导航栏/表格/表单/弹窗/3D预览/上传区 — 所有CSS变量驱动的元素

## 0.44.0 (2026-06-30)
- 新增可放置平面视觉 — 智能摆放后显示贴合区域
- 规则: 模型全在Z+侧 / Z+绿Z-蓝 / 外置偏移3mm+平行偏移2mm
- 前端: layface.js 新增 showPlacementPlane/hidePlacementPlane 清理6条路径

## 0.43.0 (2026-06-24)
- 新增"保存当前方向并报价"按钮
- 预览界面可保存智能/手动摆放后的朝向，重新切片报价
- 后端: quote接口新增 orient_x/y/z 参数
- 前端: 蓝紫色按钮, 获取 quoteOptions.orientation 重新报价

## 0.42.0–0.42.4 (2026-06-24)
- 智能摆放去掉Z轴微调
- 降低 LR 模型影响 30%→5%
- 修复评分公式: score = contact_area^1.5 × (1-overhang)² / z_height
- 智能摆放集成 PrusaSlicer 真实切片验证
- 重写智能摆放算法 — 类 OrcaSlicer 多源候选+复合评分+稳定性检查

## 0.41.0–0.41.20 (2026-06-21~24)
- 多种 UI 修复: 按钮图标清理、居中功能、颜色一致性、缩略图问题
- 行内改色快路径、颜色框定位、3D预览旋转后居中
- 切换机型后模型重新居中

## 0.40.0 (2026-06-21)
- 智能摆放面过滤重构(凸包表面筛选+法向量方向过滤)
- 缩略图/预览颜色一致、底板尺寸动态化

## 0.39.0–0.39.3 (2026-06-24)
- 朝向优化自学习系统
- 前端集成自动摆放
- 智能摆放优化(内部面过滤加强+自动居中)

## 0.38.0–0.38.5

- 前端集成自动摆放
