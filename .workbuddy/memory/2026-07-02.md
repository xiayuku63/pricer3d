### P0 修复过程中发现的额外 bug
1. **隐藏 bug**: `_safe_add_column` 的 `except: pass` 掩盖了 SQLAlchemy 2.0 兼容性问题——所有 ALTER TABLE 从未真正执行（`text()` 包装缺失）
2. **DetachedInstanceError**: `admin_list_audit` 在 session 外访问 `row.created_at`，UTCDateTime TypeDecorator 暴露了原本被 String 类型掩盖的 session 生命周期问题
3. **billing_orders 500**: 预存 bug，`PaginatedData` schema 要求 `total` 字段但路由未返回

### 关键架构决策
- TypeDecorator 优于直接改 Column 类型：无需修改 42 处写入代码
- `expire_on_commit=True`（默认）会导致 session 外 ORM 属性访问失败，需要代码审查所有路由
